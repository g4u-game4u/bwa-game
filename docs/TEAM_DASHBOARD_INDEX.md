## üõ†Ô∏è Technical Reference

### Source Code Documentation

Key service files with JSDoc comments:

- **TeamAggregateService**: `src/app/services/team-aggregate.service.ts`
  - Manages team aggregate data queries
  - Implements caching with 5-minute TTL
  - Handles error scenarios

- **AggregateQueryBuilderService**: `src/app/services/aggregate-query-builder.service.ts`
  - Constructs MongoDB aggregate queries
  - Supports Funifier relative date expressions
  - Provides query optimization

- **GraphDataProcessorService**: `src/app/services/graph-data-processor.service.ts`
  - Processes aggregate results for charts
  - Fills missing dates with zero values
  - Creates Chart.js datasets

- **TeamManagementDashboardComponent**: `src/app/pages/dashboard/team-management-dashboard/team-management-dashboard.component.ts`
  - Main dashboard component
  - Orchestrates data loading
  - Manages user interactions

### API Endpoints

- **Aggregate Endpoint**: `POST /v3/database/{collection}/aggregate`
- **Player Endpoint**: `GET /v3/player/{userId}`

See [API Integration Patterns](TEAM_DASHBOARD_API_INTEGRATION.md#funifier-api-endpoints) for details.
